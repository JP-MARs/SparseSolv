Detailed Description

Sparse Matrix Template Class このファイルはテンプレートクラス SparseMatTMPL を提供します。 内部では一時保存用の map[] を用いて要素を構築し、 必要に応じて Eigen::SparseMatrix に確定（fix）します。 行列構成時は一時保存状態で利用することが一般的です。 行列が完成した場合、fix() メソッドを呼び出して確定させます。 その後は Eigen の高速な演算が利用可能になります。
