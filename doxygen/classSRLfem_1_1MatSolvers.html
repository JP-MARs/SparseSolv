SRLfem::MatSolvers Class Reference

Linear matrix solvers for sparse matrices このファイルは疎行列の線形ソルバを提供します 加速係数付きのICCG法やMRTR法を実装しています。 収束判定処理の内部変数を持っているので、staticではなく、インスタンス化して使ってください。 More...

Detailed Description

Linear matrix solvers for sparse matrices このファイルは疎行列の線形ソルバを提供します 加速係数付きのICCG法やMRTR法を実装しています。 収束判定処理の内部変数を持っているので、staticではなく、インスタンス化して使ってください。
